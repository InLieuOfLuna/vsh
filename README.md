# VSH Server

A very simple server containing a minimal build of vsh2

# Features

* Uses the legacy airblast
* Does not (And does not plan to) have
  * Medic Revives
  * RTD
  * Taunt Crits
  * Amplifier
* Maps
  * vsh_reservoir_v2
  * vsh_weaponsdepot_final
  * arena_farm_feud_b1
